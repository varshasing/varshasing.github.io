Speech to Emotion Recognition

Using a deep learning architecture to identify the underlying emootion given some English speaking audio, formally known as Speech to Emotion Recognition (SER). Our model differentiates between anger, disgust, fear, happiness, sadness, and neutrality. We set a baseline model to base our improvements against, as a CNN. We tried a variety of architectures with a CNN with GRU, CNN-Transformer, ResNet, Mamba, and hybrid Mamba-CNN model. Our Mamba-CNN model performed best with an improvement of of 1.29% As compared to the base CNN.

Tech: Python, TensorFlow, Deep Learning, Audio Preprocessing

BeagleBone Black Traffic Light Controller

Linux kernel module that controls a model traffic light via the BeagleBone Black's GPIO pins: some pins are programmed as input (reading from buttons) and some others are as outputs (controlling LEDs). The pins are controlled entirely from kernelspace, there is no userspace component in this project. We have 3 operational modes (as well as a pedestrian call button and pedestrian mode), as well as a readable and writable character device for viewing information about the current status, and writing to change the current cycle rate. Finaly, we added a "lightbulb check", which lights up all three traffic lights as long as both buttons are held down simultaneously.

Tech: Kernel programming, BeagleBone Black, C, Circuit-building

Jailbreaking Large Language Models

Assessed the scope and severity of jailbreaking in commonly used models and proposed a practical workflow to support safer interactions with LLMs. We conducted a comparitive analysis of jailbreak vulnerabilities across severaal leading chatbot platforms, evaluating their susceptibility to prompt-based attacks. We quantify the success rate of jailbreaking attempst and categorizr the tyeps of content or actions that emerge from these exploits. Finally, we developed a modular defense framework utilizing Llama-guard that integrates prompt validation and response filtering to reduce the likelihood of unsafe outputs.

Tech: Git, Python, APIs, React
