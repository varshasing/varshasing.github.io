BeagleBone Black Traffic Light Controller

Linux kernel module that controls a model traffic light via the BeagleBone Black's GPIO pins: some pins are programmed as input (reading from buttons) and some others are as outputs (controlling LEDs). The pins are controlled entirely from kernelspace, there is no userspace component in this project. We have 3 operational modes (as well as a pedestrian call button and pedestrian mode), as well as a readable and writable character device for viewing information about the current status, and writing to change the current cycle rate. Finaly, we added a "lightbulb check", which lights up all three traffic lights as long as both buttons are held down simultaneously.

Tech: Kernel programming, BeagleBone Black, C, Circuit-building

File Allocation Table-based File System

Implemented a simple file system on top of a virtual disk. Implementing a library that offers a set of basic file system calls (open, close, create, delete, read, write, get file size, list files, lseek, truncate) to applications. The virtual disk has 8192 blocks, each holding 4KB (32 MiB), with a maximum file size of 16 MiB. File system implemented usiing FAT design, with meta-information blocks for the super block and directory, and storing file descriptors. Optimized partition to store 31.2 MiB of file content, compared to required 16 MiB.

Dual-Tone Multi-Frequency Decoder

Matlab study in Dual-Tone Multi-Frequency (DTMF) as used in phone dialing, and is usd worldwide for telephone signaling to the call switching center in the voice frequency band. When the input to a filter bank is a DTMF signal, the output from two of the bandpass filters should be large and the rest should be small. If two outputs are large, then it is understood to be the corresponding frequencies. This project has two sections. The first focuses on generating telephone keypad tones, analyzing telephone keypad tones with the Fourier Transform, and designing a telephone keypad decoder. The second section analyzes and designs a simple bandpass filter to be used in the filter bank of a DTMF decoder, and creates and tests a complete DTMF decoding system.

Tech: Matlab, Signal Processing

Tasked to build a Blackjack agent with reinforcement learning to achieve as close to published win rates as possible.
